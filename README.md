# MoonShineR_package
## Updated June 4, 2026

This site hosts the source code for the MoonShineR package.

Published in CRAN repository as v1.1.0
https://cran.r-project.org/web/packages/MoonShineR/index.html

For further details, see MoonShine suite homepage: https://github.com/Crampton-Lab/MoonShine

Citation: Poon, L., Jenks, I. T., & Crampton, W. G. (2024). MoonShine: A software-hardware system for simulating moonlight ground illuminance and re-creating artificial moonlight cycles in a laboratory environment. _Methods in Ecology and Evolution_, _15_(4), 701-715. https://doi.org/10.1111/2041-210X.14299
